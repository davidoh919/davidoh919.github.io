What is Backend Development?

Backend is the part of a computer system or application that is not directly accessed by the user, typically responsible for storing and manipulating data. In general, backend development stands for server-side development of a website. Main Topics

Internet
General knowledge how the Internet works. Includes concepts of HTTP, Browsers, DNS, hosting, Web sockets and security
Frontend Basics
Basic knowledge on fontend of a web page. Includes concepts of HTML, CSS, Javascript
OS
General knowledge on operating system. Includes concepts of Terminal usage, process management, threads(concurrency), memory management, interprocess communication, I/O management, POSIX basics, and basic networking concepts
Version Control System
Basic on Git and repository hosting service (ex. Github)
Databases
General knowledge on relational databases (ex. MySQL), NoSQL (ex. MongoDB), database normalization, and N+1 problems
APIs
General knowledge on API and widely used APIs such as REST, JSON APIs, ands API related to authentication (ex. Basic Authentication, Token Authentication, OAuth)
Design and Development Principles
General knowledge of design principles for software engineering, including SOLID, KISS, YAGNI, DRY
Architectural Patterns
General knowledge on software architectures which work as reusable solution, including concpets of Monolithinc Apps, Microservices, SOA, Serverless
